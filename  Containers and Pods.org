# kubernetes

#####
#Pod# 
#####
are the smallest and most basic building blocks of the Kubernetes model
a pod consists of one or more containers , storage resources , and an unique IP address in the Kubernetes cluster network.

Here are the commands used in this lesson:
###################################################
#1.Create a simple pod running an nginx container:#
###################################################

cat << EOF | kubectl create -f -
apiVersion: v1
kind: Pod 
metadata:    
  name: nginx
spec:
  containers:   
  - name: nginx
    image: nginx
EOF
-----------------------------------------------------------------------------------

##################################################################################
#2.Get a list of pods and verify that your new nginx pod is in the Running state:#
##################################################################################
kubectl get pods
-----------------------------------------------------------------------------------

###################################################################################
#3.Get more information about your nginx pod:                                     #
###################################################################################
kubectl describe pod nginx
-----------------------------------------------------------------------------------

####################
#4.Delete the pod: #
####################
kubectl delete pod nginx
